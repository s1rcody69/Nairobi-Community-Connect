# Nairobi Community Connect

## Project Overview
Nairobi Community Connect is a community-based website designed to help connect residents in Nairobi with support services, volunteer opportunities, and local events. The goal of this project is to promote collaboration and strengthen local communities.

## Features
- Home page with introduction and call to action
- Find Help page with support categories
- Volunteer registration form
- Events page with upcoming community events
- About page explaining the mission and vision
- Stories page for testemonials
- Contact page for communication

## Technologies Used
- HTML5
- Tailwind CSS (CDN)
- Custom CSS
- Google Fonts

## Purpose
This project was created as a capstone assignment to demonstrate my understanding of HTML structure, layout design, forms, and basic styling using Tailwind CSS and Custom CSS.
